# CineMatch

Movie Recommendation System menggunakan Singular Value Decomposition (SVD).

## Model Performance

- RMSE: 0.8635
- MAE: 0.6624
- Precision@10: 63.36%

## Features

- Personalized Recommendation
- User Rating History
- Exploratory Data Analysis
- CSV Export

## Dataset

MovieLens Latest Small

100,836 ratings
610 users
9,742 movies

## Run Locally

pip install -r requirements.txt

streamlit run app.py
